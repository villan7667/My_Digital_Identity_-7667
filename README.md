## 🛠️ The Tech Stack
This portfolio isn't just about looks; it's about clean, efficient execution.

* **Frontend:** HTML5, CSS3 (Custom Properties & Keyframes)
* **Interactivity:** Vanilla JavaScript (Dynamic Greetings, Typed.js)
* **Aesthetic:** Glitch effects & Glassmorphism
* **Deployment:** Netlify

## 🚀 Key Features

### 🕒 Real-Time Adaptation
The hero section automatically detects the visitor's local time to provide a personalized greeting:
- `05:00 - 11:59` → **Good Morning**
- `12:00 - 17:59` → **Good Afternoon**
- `18:00 - 04:59` → **Good Evening**

### 🎨 Glitch UI
Uses CSS `clip-path` and `pseudo-elements` to create a "cyber-villain" glitch effect on the main heading, ensuring the brand identity is felt the moment the page loads.
